# Project: React Tutorial - Personal Finance Tracker

## Overview
A comprehensive hands-on React.js tutorial that teaches modern React development by building a complete Personal Finance Tracker application from scratch.

## Tech Stack
- **Framework**: React 18+ with TypeScript
- **Build Tool**: Vite
- **Routing**: React Router v6
- **Server State**: TanStack Query v5
- **Client State**: Zustand
- **Forms**: React Hook Form + Zod
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Mock API**: json-server
- **Icons**: Lucide React

## Conventions
- **Tutorial Format**: Markdown with code blocks, explanations, and exercises
- **Code Style**: ESLint + Prettier with Airbnb config
- **Commits**: Conventional commits (feat:, fix:, docs:)
- **Branch per Module**: Each module can be a git branch for easy navigation

## Target Audience
- Developers with basic JavaScript knowledge
- Some HTML/CSS experience
- No prior React experience required
